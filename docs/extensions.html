Extensions

Extensions hold templates, setup information, code and assets (JS, CSS, Images). An extension can be added to add additional functionality. The "Core" extension provides the basic system functionality.

Adding an extension

Add the folder into the root of the app, along side the "extension.Core" folder. Edit config.php and add the extension.

Extensions can provide assets (JS, CSS, Images) and you may need to run the build scripts to copy the right assets into the web folders.
