Release v2.0.0, 25th Aug 2017

Source & Download

Note the minimum requirements have been raised to PHP 7.0!

There have not been many functional changes - but this release upgrades a lot of the libraries we use and updates our code to accomdate that. Probably imcomplete lists are below:

Functionality:

  • Remove Reason For Closing account #738
  • CLI script to run one particular importer now, regardless of when it was last run #730

UI:

  • Remove username option from login form #734
  • Relabel "Monthly" as "Calendar" #735
  • Import UI - add more messages to try and make process clearer #733
  • List imports on group, show status in list (eg "enabled", "disabled") #732
  • Fixing width of pages #684
  • Include Existing Group Search in New Event Wizard #719

Fixes:

  • Recurring Events in an area (not venue) does not copy the area! #739
  • pass new EventFilterParams options properly! #736
  • Don't allow 2 importers with the same URL #731
  • If trying to import something, and get 500 error, whole task crashes. #711
