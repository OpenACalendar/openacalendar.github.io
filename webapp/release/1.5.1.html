Release v1.5.1, 17th September 2015

Source

This release included some things to help fight spam, including some new features ...

  • Verifying a user email should require a user action not just a link #437
  • Store IPs of user actions #434

As part of this, many improvments and bugfixes to the sysadmin interface.

  • Sysadmin can't undelete group #413
  • Sysadmin comments on users #431
  • Sysadmin - when looking at user, can see things edited #429
  • Sysadmin can't change parent area of area - bug fix #440

And also, some extension hooks so actions can happen there.

  • Extension call back for after creating new user #432
  • More Extension hooks, beforeGroupSave and beforeAreaSave like beforeVenueSave #435

  • Add Google Analytics Options #430
