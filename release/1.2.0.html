Release v1.2.0, ??????

New Features:

  • User Notifications in Web pages. #169
  • Editing recurring events. #37
  • Events can be tagged, optional feature per site.
  • New Facebook importer, better Meetup importer. These are new extensions that must be enabled, see the docs. #177 & #182
  • 1 Event in many groups - there is now a user accesible UI for this, previously only system administrators could do this. #34
  • Can search list of events for free text like you can with groups.
  • Import URL History should appear in main site history. #206
  • Events now have an Ticket URL as well as an information URL.
  • ICAL feeds have new LOCATION and GEO fields.
  • Schema.org markup.
  • Better duplicate event detection using URLs. #121
  • Users can delete venues from normal UI. #140
