Release v1.5.0, DUE SOON

Key improvements

  • Any users can create and edit tags, if they have the right permissions #392 (Previously only admins could)
  • When making new event in group, check for dupes #371
  • Event Custom Fields, single and multi line text #374
  • New feature per site - when making a change, user can leave comment #369
  • Extensions can add content at key points in system #342
  • Reports for sys admins #344
  • Combine notify watch emails into one per site and per user #327

An incomplete list of minor improvements

  • Add meta tags to our pages so future user agents can explore page structure #382
  • Incoming Webmentions and Pingbacks are logged #324
  • Admins can rename user groups #370
  • Deleted and Cancelled events more obvious #378
  • Share Area Link #337
  • Event page breadcrumbs shows group - but if no groups show location breadcrumbs instead #360
  • When searching for duplicate events, can specify a list of words not to match using new config "findDuplicateEventsNoMatchSummary" #353
  • Events coming up email includes cancelled ones #386
  • On mobile phones, show hide option to reduce screen clutter and better click zones #383 #384
  • When user adding or editing event, show nice date range text #372
  • On area show prompt to register that makes you follow at once #330
  • Show Media in event popup on Calendar #302
