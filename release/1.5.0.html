Release v1.5.0, DUE SOON

Note we have made our own task system. The cron jobs you have running will have to be overhauled, but the good news is this should be the last time you will have to do that.

  • New event Wizard #375
  • Any users can create and edit tags, if they have the right permissions #392 (Previously only admins could)
  • When making new event in group, check for dupes #371
  • Event Custom Fields, single and multi line text #374
  • New feature per site - when making a change, user can leave comment #369
  • Extensions can add content & features at key points in system #368 #340 #341 #342 #334
  • Reports for sys admins #344
  • Combine notify watch emails into one per site and per user #327
  • Make our own task system #325
  • Users can follow areas #328
  • Logged out user can see "add event" and "add group", if they would have that permission if they were logged in and verified #335 #408
  • Restore rollback to old version of event links #365 #399
  • If no results found when listing events, better error messages #373
  • CSV Export #289
  • If click edit venue on an event with an existing venue, offer more options in UI on exactly what user wants to do. #359
